# obake 👻 — ガイスター (Geister)

ボードゲーム「ガイスター」と全く同じルールで遊べる、ブラウザ完結の2人対戦ゲームです。
インストール不要・ビルド不要。`index.html` を開くだけで遊べます。

## 遊び方

`index.html` をブラウザで開くだけです。

```
# 例: ローカルでそのまま開く
open index.html            # macOS
xdg-open index.html        # Linux

# もしくは簡易サーバ経由で
python3 -m http.server 8000   # → http://localhost:8000
```

同一画面での2人対戦（ホットシート）です。2つのモードから選べます。

### モード

- **通常モード**: 手番交代のたびに目隠し画面を挟み、相手にコマの色を見せません。端末を相手に渡してプレイします。
- **記憶モード（COOL）**: コマに 1〜8 の番号を振り、対局中はコマの色を一切表示しません。開始時に「自分のどの番号が青／赤か」を覚えるだけ。対局中は番号トークン（自分=シアン、相手=マゼンタのネオンリング）だけが表示されるので、**手番ごとの目隠しが不要**です。色を記憶して戦うモードです。
  - 開始時の配置と暗記のときだけ、各プレイヤーが交代で1回ずつ端末を見ます（その2回だけは目隠しを挟みます）。

## ルール

- **盤面**: 6×6 のマス。
- **コマ**: 各プレイヤーは **青（良いオバケ）4体** と **赤（悪いオバケ）4体** の計8体を持ちます。
- **配置**: ゲーム開始時、自陣の2行（中央4列）にコマを並べ、どれを青／赤にするかを各自が秘密に決めます（青はちょうど4体）。
- **情報**: 自分のコマの色だけが分かり、相手のコマの色は分かりません（捕獲したときに判明します）。
- **移動**: 1手番につき、自分のコマ1体を上下左右いずれかに1マス動かします。味方のコマがいるマスには進めません。相手のコマがいるマスに進むと、そのコマを捕獲します。
- **勝利条件（いずれかを満たせば勝ち）**:
  1. 相手の**青**を4体すべて捕獲する。
  2. 自分の**赤**を4体すべて相手に捕獲させる（＝相手に赤を全部取らせる）。
  3. 自分の**青**を、相手陣の左右いずれかの**隅マス（脱出マス）**から盤外へ脱出させる。

### 操作

- **配置フェーズ**: 自陣のコマをクリックして青にするコマを4体選び、「この配置で確定」。
- **対局フェーズ**: 自分のコマをクリックして選択 → ハイライトされた移動先をクリック。
  - 赤い枠は捕獲できる相手コマ、紫のドットは空きマスへの移動です。
  - 自分の青コマが脱出マス（縞模様の隅）にいるときは、そのコマをもう一度クリックすると脱出できます。
- **プライバシーモード**（既定ON）: 手番交代のたびに目隠し画面を挟み、相手にコマの色が見えないようにします。
